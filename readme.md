# Sugar
Realtime Mastodon Client built with Nuxt.js and Electron.
## Development
### yarn
```
yarn dev
```
or
```
yarn start
```
### npm
```
npm run dev
```
or
```
npm start
```

## Unit test
### yarn
```
yarn test
```
### npm
```
npm run test
```

## Lint
### yarn
```
yarn lint
```
### npm
```
npm run lint
```

## Format
### yarn
```
yarn fmt
```
### npm
```
npm run fmt
```

## Packaging
### yarn
```
yarn package
```
### npm
```
npm run package
```

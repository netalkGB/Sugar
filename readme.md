# Sugar
Realtime Mastodon Client built with Nuxt.js and Electron.  
![screenshot](https://user-images.githubusercontent.com/13611993/129604705-001e50ef-8965-45b3-84df-ad62552a9196.png)
## Development
### yarn
```
yarn dev
```
or
```
yarn start
```
### npm
```
npm run dev
```
or
```
npm start
```

## Unit test
### yarn
```
yarn test
```
### npm
```
npm run test
```

## Lint
### yarn
```
yarn lint
```
### npm
```
npm run lint
```

## Format
### yarn
```
yarn fmt
```
### npm
```
npm run fmt
```

## Packaging
### yarn
```
yarn package
```
### npm
```
npm run package
```
